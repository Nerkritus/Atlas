AtlasUtil
--------------------------

This library contains various helpful utility methods used throughout the Atlas Engine.

Graphics
----
ImageLoader helps load image files. Supports bitmap, jpeg and png

IO
----
AtlasLog - Supports logging out to a text file. Useful for debugging games and the engine based upon the c++ STL sstream

Math
----
Random - Provides random number generation functions based upon the C++ STL random

Time
----
AtlasStopwatch - Provides high precision timing functions based upon the C++ STL chrono
